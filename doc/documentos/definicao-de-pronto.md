# InboxFlow — definição de pronto

Uma fatia do projeto só está pronta quando:

- [ ] o objetivo foi atendido
- [ ] o escopo foi respeitado
- [ ] a solução está legível
- [ ] a mudança é pequena e revisável
- [ ] existe validação mínima
- [ ] riscos residuais foram apontados
- [ ] o próximo passo está claro
