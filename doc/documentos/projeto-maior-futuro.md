# InboxFlow — evolução futura / projeto maior

## Visão

O InboxFlow pode evoluir para um produto mais robusto de operação e triagem multi-canal.

## Caminho futuro

- integração real com WhatsApp
- múltiplos canais
- workflows automáticos
- fila por time/responsável
- resposta assistida
- histórico de atendimento
- analytics mais fortes

## Regra

Essa visão futura não deve poluir o MVP atual.

## Uso estratégico

- o MVP rápido gera conteúdo e portfólio
- a visão maior aumenta percepção de ambição e roadmap
