# InboxFlow — protocolo operacional

## Regra principal

Não pedir o projeto inteiro de uma vez para a IA.

## Regras obrigatórias

- planejar antes de editar
- trabalhar em fatias pequenas
- validar antes de chamar de pronto
- evitar overengineering
- respeitar o fora de escopo
- manter o WhatsApp como conector futuro no MVP

## Formato padrão de resposta esperado da IA

1. Entendimento
2. Plano
3. Execução
4. Validação
5. Próximo passo
