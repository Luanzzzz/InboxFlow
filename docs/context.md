# InboxFlow

Data de criação: 2026-04-07
Status: planejamento inicial

## O que é

InboxFlow é um sistema Django + IA + automação para triagem e organização operacional de mensagens/demandas.

## Objetivo

Construir um projeto pequeno, postável e útil de verdade, que mostre:

- Django
- IA aplicada
- automação operacional
- modelagem de domínio
- arquitetura clara para desenvolvimento assistido por IA

## Tese do produto

Muitas operações pequenas/médias não têm uma fila clara para tratar mensagens, demandas e atendimentos.
O resultado é:

- urgências misturadas
- mensagens perdidas
- baixa prioridade ocupando espaço
- falta de visão operacional

InboxFlow existe para transformar entradas espalhadas em uma fila operacional organizada.

## Papel do WhatsApp no MVP

O sistema deve nascer preparado para aceitar integração futura com WhatsApp, mas no MVP a conexão real fica como **opção futura**, não como dependência obrigatória.

## Uso estratégico

Este projeto serve como:

- ativo de portfólio
- prova de domínio em Django
- demonstração de IA aplicada a problema real
- combustível para posts da próxima semana
